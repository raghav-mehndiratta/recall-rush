# Recall Rush

Recall Rush is a web-based memory game platform where users can choose from multiple fun brain-training games to test and improve their memory skills.

## Features

- Multiple memory games (currently supports 2 -> Sequence Memory, Verbal Memory)
- Simple and clean UI
- Score tracking
- Leaderboard stored in browser (localStorage)

## How To Play

1. Open the website
2. Choose any memory game you want
3. Read the instructions shown on the screen
4. Press start
5. Improve your score every round

## Tech Stack

- HTML
- CSS
- JavaScript (ES6 classes)
- LocalStorage for leaderboard

## Future Improvements

- Add more memory games
